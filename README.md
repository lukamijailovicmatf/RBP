# RBP
Repozitorijum za materijale iz kursa Relacione baze podataka za I - smer na Matematickom fakultetu

NAPOMENA: ako se bilo koji pdf ne otvara u okviru Github-a, samo ga lokalno preuzeti sa Github-a na Desktop ili bilo koji drugi direktorijum po zelji i onda ce biti dostupan za dalju upotrebu.

Linux OS: Ctrl + Alt + T => cd Desktop/ => git clone https://github.com/lukamijailovicmatf/RBP.git.
Nakon komande "git clone https://github.com/lukamijailovicmatf/RBP.git" iz terminala i njenog zavrsetka citav repozitorijum ce se nalaziti lokalno na Desktop-u i svi folderi / fajlovi / pdf-ovi ce biti dostupni za dalju upotrebu.
